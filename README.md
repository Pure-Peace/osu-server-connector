### Scripts

**Dev app**

```bash
yarn dev
```

**Build app**

```bash
yarn release
```

**Build web resources**

```bash
yarn build
```

**Dev web**

```bash
yarn start
```

**Serve `/build`**

```bash
yarn serve
```
